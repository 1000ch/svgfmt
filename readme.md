# svgfmt ![test](https://github.com/1000ch/svgfmt/actions/workflows/test.yml/badge.svg)

Format SVG string.

## Install

```sh
npm install svgfmt
```

## Usage

```javascript
const svgfmt = require('svgfmt');

svgfmt('<foo>Foo!</foo>');
```
