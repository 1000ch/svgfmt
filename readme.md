# [xmlfmt](https://1000ch.github.io/xmlfmt/)
[![Build Status](https://travis-ci.org/1000ch/xmlfmt.svg?branch=master)](https://travis-ci.org/1000ch/xmlfmt)

Format XML string.

## Usage

```javascript
import xmlfmt from 'xmlfmt';

xmlfmt('<foo>Foo!</foo>');
```
