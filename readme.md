# [svgfmt](https://1000ch.github.io/svgfmt/) [![Build Status](https://travis-ci.org/1000ch/svgfmt.svg?branch=master)](https://travis-ci.org/1000ch/svgfmt)

Format SVG string.

## Usage

```javascript
const svgfmt = require('svgfmt');

svgfmt('<foo>Foo!</foo>');
```
