# svgfmt [![Build Status](https://travis-ci.org/1000ch/svgfmt.svg?branch=master)](https://travis-ci.org/1000ch/svgfmt)

Format SVG string.

## Install

```sh
npm install svgfmt
```

## Usage

```javascript
const svgfmt = require('svgfmt');

svgfmt('<foo>Foo!</foo>');
```
